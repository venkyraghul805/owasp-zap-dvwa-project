# owasp-zap-dvwa-project
Web Application Security Testing project using OWASP ZAP and DVWA. Includes vulnerability scanning, authenticated testing, and manual exploitation of SQL Injection and Cross-Site Scripting (XSS) along with security misconfiguration analysis.
# 🔐 OWASP ZAP DVWA Project

## 📌 Overview
This project demonstrates **Web Application Security Testing** using **OWASP ZAP** on **DVWA (Damn Vulnerable Web Application)**.

The objective is to identify and analyze common web vulnerabilities such as:
- SQL Injection
- Cross-Site Scripting (XSS)
- Security Misconfiguration

---

## 🛠 Tools & Technologies
- OWASP ZAP
- DVWA (Damn Vulnerable Web Application)
- Docker
- Kali Linux / Windows
- Git & GitHub

---

## ⚙️ Project Setup
1. Install Docker and run DVWA container  
2. Configure OWASP ZAP proxy  
3. Intercept and analyze web traffic  
4. Perform automated and manual testing  

---

## 🔍 Vulnerabilities Tested
- SQL Injection  
- Cross-Site Scripting (XSS)  
- Security Misconfiguration  

---

## 📊 Results
- Identified multiple vulnerabilities using ZAP  
- Generated automated scan reports  
- Demonstrated manual exploitation techniques
